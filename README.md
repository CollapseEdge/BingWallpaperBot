# BingWallpaperBot

本仓库每天会从bing爬取壁纸，并储存在data.json中

> 🔨 `Bing 壁纸仓库`上阵啦 😄

<small>tips:在url前加`bing.com`就可以使用了</small>

----

### 坑边闲话：

> 创建这个项目的理由是网上的许多随机图片api的内容参差不齐，且不能稳定使用,
>
> 看到网上有大佬利用GitHub Action制作Bing壁纸接口，便有了自己动手写一个的想法

### Final_Target:

> 本项目的目标是打造一款图片质量可靠，且稳定的随机图片api
>
> 优点：
>
> - 图片来源是微软bing，质量可靠
> - 图片也储存在bing，稳定快速
